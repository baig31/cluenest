# Product Screen Design
Version: 1.0

---

# Goal

The Product Screen should allow an administrator to create a complete Product Profile in under 10 minutes.

The screen should be simple, organized, and require no duplicate work.

---

# Layout

The Product Screen will be divided into logical sections.

1. General Information
2. Images
3. Specifications
4. Product Insights
5. Affiliate Links
6. SEO
7. Publish

---

# Section 1 - General Information

Fields

- Product Name
- Brand
- Category
- Model Number
- Product Status
- Short Description
- Long Description

---

# Section 2 - Images

Fields

- Primary Image
- Product Gallery

Future

- Drag & Drop Upload
- Image Compression
- Auto Alt Text

---

# Section 3 - Specifications

Purpose

Every category has different specifications.

Specifications must be flexible.

Example

Capacity

↓

7 Kg

Motor

↓

Inverter

Power

↓

230V

Warranty

↓

2 Years

The administrator can add unlimited specifications.

# Golden Rule

Every Product Profile is the Single Source of Truth.

Updating the Product Profile updates every Buying Guide, Comparison, Brand Page, and Category Page automatically.
